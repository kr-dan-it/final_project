Hello. This project uses the Extended Iris Dataset from Kaggle. The description from the Kaggle site linked above reads the following:

This CSV file contains an extended version of the classic Iris dataset, enriching it with additional attributes and features. The dataset consists of 1200 rows and 21 columns that include both original attributes like 'Species', 'Sepal Length', 'Sepal Width', 'Petal Length', and 'Petal Width', as well as newly added environmental and morphological attributes such as 'Elevation', 'Soil Type', and 'Petal Curvature'.


Out of the new attributes, I used Elevation, Soil Type, Petal Curvature, and Leaf Area. The page was styled using Bootstrap and Crispy Forms.
